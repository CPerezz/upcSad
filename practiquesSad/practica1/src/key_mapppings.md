# Key mappings for ReadLine Tecla -> int[]

**`int[]` is really a `char[]` formatted.**

```
- RIGHT_ARROW => 27 91 67
- LEFT_ARROW => 27 91 68
- UPPER_ARROW => 27 91 65
- DOWN_ARROW => 27 91 66
- BACKSPACE => 127
- ENTER => 13
- ESC => 27
- Ctrl + C => 3
- Ctrl + X => 24
- Space => 32
- Insert => 27 91 50
```

**SUPR gives the precious: `[2312611111191127 91 51`**
